# Performance Quick Capture Template

Use this as a reusable template for converting rough daily or weekly notes into concise, impact-focused performance entries.

## Metadata

- Tool: ChatGPT
- Version: v1.0

## Overview

This template is designed for fast logging of visible and invisible work for impact tracking.

Use it when rough notes need to be converted into concise performance documentation that highlights ownership, influence, initiative, and cross-team value without exaggeration.

## Instructions

I'm logging work for visibility and performance documentation.

Below are rough notes from today or this week.

Convert this into concise, impact-focused performance entries.

For each meaningful contribution, extract:

- **Work Item Title**
- **What I Actually Did**
- **Why It Mattered**
- **Who It Impacted**
- **Outcome / Value Created**
- **Visibility Opportunity** (how this could be communicated upward or outward)

Emphasize ownership, influence, initiative, and cross-team value.  
Avoid fluff. Avoid exaggeration. Make impact clear.
