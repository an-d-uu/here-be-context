# Performance Monthly Visibility Synthesis Template

Use this as a reusable template for translating individual performance entries into a strategic monthly visibility narrative.

## Metadata

- Tool: ChatGPT
- Version: v1.0

## Overview

This template is designed to synthesize multiple performance entries into a higher-level monthly visibility summary.

Use it when individual entries need to be analyzed for recurring strengths, under-communicated impact, and leadership-facing value framing.

## Instructions

Based on the performance entries below:

1. Identify recurring strengths and themes.
2. Identify types of impact that may be undervalued or under-communicated.
3. Summarize my overall value proposition in 5 to 7 sentences suitable for leadership visibility.
4. Suggest 3 ways I can make this impact more visible next month without adding new work.

Be analytical and honest.
