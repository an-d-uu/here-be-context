# Dialogue Check Grimoire

## Overview

This file contains a reusable speech-control reference for shaping professional, clear, and scope-safe responses.

Use it when you want the model to:

- reduce implied commitments
- keep tone neutral and controlled
- choose phrasing that stays professional under pressure
- avoid expanding ownership, scope, or timelines unintentionally
- pull from approved wording sources when a supporting lexicons is available

This is not a template pack.  
This is not a one-off project instruction.  
This is a reusable speech discipline reference.

It is intended to be saved as a standalone source file and included as reference context when writing, revising, or generating responses.

## Lexicon Use

When relevant lexicon files are available in the project source, use them as the preferred source for controlled phrasing, approved substitutions, stylistic constraints, activation rules, and optional themed language.

If multiple lexicons are available:

- apply only those that are relevant to the request and active context
- preserve clarity, professionalism, and scope safety when combining them
- prefer the most specific lexicon when guidance overlaps
- allow a more specific mode lexicon to override a more general lexicon when both are explicitly active

---

## Notes

- Prefer clarity over cleverness.
- Avoid implied ownership unless explicitly intended.
- Avoid timelines unless confirmed.
- Neutral language scales better than precise promises.

Words matter.  
Use them like they might be quoted later.
